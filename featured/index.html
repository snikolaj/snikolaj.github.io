Projects which I particularly enjoyed, am proud of, or are unique

My featured projects:

  • DIY Graphics Card

    A completely DIY graphics card based on an extremely optimized version of the popular Ben Eater design. The ATMega32 can be programmed to do all kinds of geometric calculations and rendering, while the PALs enable for different resolutions based on the crystal used.

  • DIY USB MIDI Converter

    A DIY MIDI Converter based on the PIC16F1459 microcontroller, handles SysEx like a champ and does rudimentary MIDI filtering for incoming notes.

  • ProgramKitsForEveryone

    A project which contains a fully contained project and documentation for beginners to teach themselves programming games in one single place.

  • LibreOffice Calc BASIC Tutorial

    Code, spreadsheets and documentation used for the LibreOffice BASIC tutorial series.

  • YM64Axx Research

    Translated and redesigned datasheet and research for a relatively obscure Japanese-only Yamaha melody chip.

  • AP Statistics TI-84 Programs

    A small set of programs which I used to help me with AP Statistics. The most useful one is Inference. Written in TI-BASIC.
